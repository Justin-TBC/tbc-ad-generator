Meta Ad Library

Search competitor ads directly from References → Competitors tab. Requires a Facebook access token with ads_read permission.

1. Go to developers.facebook.com → create an app (type: Business)
2. Add the Marketing API product
3. In Graph API Explorer, generate a User Token with ads_read scope
4. Set META_ACCESS_TOKEN in Railway environment variables
